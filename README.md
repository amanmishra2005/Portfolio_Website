## 💫 About Me

I'm a **Computer Science & Engineering** student at **Shri Mata Vaishno Devi University (SMVDU)**, focused on building scalable web applications and strengthening my grip on core CS fundamentals. I like sitting at the intersection of clean design and efficient system logic.

### 🚀 Currently Learning & Building

| Area | Focus |
|---|---|
| 📊 **Advanced DSA & Optimization** | Graphs, Dynamic Programming, Trees, and Recursion in Java & C — writing low-latency, optimized code |
| ⚡ **Backend & Systems Engineering** | Production-grade DB design, RESTful APIs, server performance, secure JWT auth |
| 📐 **TypeScript** | Fully type-safe code for robust, enterprise-ready systems |
| ⚛️ **Modern Frontend** | Lightning-fast SPAs with **Next.js** and Server-Side Rendering |
| 🌐 **Real-time Event Web** | Bi-directional data pipelines with **Socket.io** — live chat, sync systems, dashboards |

### 🤝 Let's Collaborate

- Open to collaborating on **Open Source MERN/Next.js applications**, backend API design, and real-time chat/collaboration tools.
- 💬 **Ask me about:** Java, C, React, Node.js, Express, MongoDB, DOM manipulation.
- ⚡ **Fun fact:** I love a good debugging mystery — nothing beats shaving milliseconds off execution time.
